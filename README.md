# Programacion con Python: 15 Proyectos de Desarrollo y Automatización

## Resumen Ejecutivo del Repositorio

Este portafolio demuestra el dominio de Python en un total de **15 proyectos completos**, cubriendo desde los fundamentos hasta la aplicación de tecnologías avanzadas.

Las habilidades principales demostradas se centran en:

* **Desarrollo Web Back-end:** Flask, Django, creación de APIs REST.
* **Ingeniería de Software:** Aplicación de Programación Orientada a Objetos (OOP) y análisis de algoritmos.
* **Data Handling & Automatización:** Integración con bases de datos (SQLAlchemy, SQLite) y consumo de APIs externas.

### Stack Tecnológico Clave

| Habilidad | Herramientas Clave |
| :--- | :--- |
| **Frameworks Web** | Django, Flask |
| **Persistencia de Datos**| SQLAlchemy, SQLite |
| **Diseño de Código** | OOP, Merge Sort |
| **Integración** | Requests, BeautifulSoup |
| **Automatización** | Smtplib |

## Guía de Instalación y Uso

Para ejecutar cualquier proyecto, se requiere **Python 3.x** y **Git**.

### Clonación

```bash
git clone [https://github.com/ciroduro01/Programacion_con_Python](https://github.com/ciroduro01/Programacion_con_Python)
cd Programacion_con_Python

```

### Ejecución

1. **Entorno Virtual:** Se recomienda crear y activar un entorno virtual.
2. **Dependencias:** Navega a la carpeta del proyecto e instala las librerías específicas (si existe un `requirements.txt`) usando: `pip install -r requirements.txt`.
3. **Lanzamiento:** Ejecuta el archivo principal (ej. `python main.py` o siguiendo las instrucciones del `README.md` interno).
