# Programacion_con_Python
## Resumen Ejecutivo
Este repositorio consolida 15 proyectos completos que demuestran el dominio de Python, desde los fundamentos de la sintaxis y estructuras de datos hasta la aplicación de tecnologías avanzadas como el desarrollo web (Full-Stack con Django) y la interacción con bases de datos relacionales y APIs.

El portafolio está diseñado para ilustrar una progresión técnica clara, cubriendo habilidades esenciales en Automatización, Data Science, Desarrollo de APIs REST y Arquitectura Web Profesional.
